# 入侵蟻：蟻巢潛入戰 (Intruder Ant: Nest Infiltration)

一款基於 HTML5 Canvas 與純 JavaScript 打造的 2D 網格制隱蔽策略網頁遊戲。玩家將扮演一隻潛入敵方蟻穴的入侵蟻，利用費洛蒙引開敵方兵蟻，尋找並摧毀深處的蟻后。

## 🎮 遊戲特色

*   **2D 網格制隱蔽探索**：在隨機生成的蟻洞中潛行，走每一步都需深思熟慮。
*   **動態費洛蒙引誘系統**：主角可隨地排放具有揮發時效的費洛蒙，引誘並誤導巡邏中或追擊中的兵蟻改變路線。
*   **Braid 織網迷宮演算法**：地圖具備豐富的分岔路口與繞道環路，消除單一路徑，提供多樣的潛行逃脫策略。
*   **真實戰爭迷霧 (Fog of War)**：採用 Bresenham 射線檢測，實現真實的視野阻擋與漆黑的探索感。
*   **多階段 Boss 戰**：蟻后周圍由 3 個氣味防護節點保護，玩家必須潛入節點並用費洛蒙充能，才能解除防護罩擊敗蟻后。
*   **精緻霓虹美學**：泥土質感迷宮搭配發光的費洛蒙粒子、兵蟻紅色警告圈與蟻后紫色防護罩，提供極佳的視覺回饋。

## 🕹️ 操作說明

*   **移動**：使用 `W`、`A`、`S`、`D` 或 **方向鍵**（每次移動一格）。
*   **釋放費洛蒙**：按下 **空白鍵 (Space)**。
    *   在一般通道：放置費洛蒙粒子以引開兵蟻。
    *   在蟻后巢穴旁的 **1、2、3 節點** 上：消耗能量為防護節點進行充能。
*   **補充能量**：收集散落在地圖上的黃色 **糖分結晶**，可立即恢復大量費洛蒙能量。
*   **勝負判定**：
    *   **勝利**：站在 3 個防護節點上各按下 4 次空白鍵（使其充能至 100%）以解鎖防護罩，最後上前接觸蟻后。
    *   **失敗**：被兵蟻撞擊 3 次（生命值歸零）。

## 🛠️ 開發架構

遊戲完全採用 Vanilla JS + CSS3 + HTML5 實作，無須任何前端框架，檔案結構如下：

*   `index.html`：遊戲主頁面與 HUD 資訊面板。
*   `index.css`：暗系泥土風搭配螢光霓虹色彩的毛玻璃 (Glassmorphism) 樣式表。
*   `js/map.js`：處理隨機迷宮生成 (DFS + Braid 織網)、費洛蒙矩陣衰減與戰爭迷霧射線檢測。
*   `js/entities.js`：角色類別定義、A* 尋路演算法、兵蟻 AI 狀態機控制。
*   `js/renderer.js`：Canvas 畫布渲染、粒子系統與發光特效繪製。
*   `js/game.js`：主遊戲控制器、遊戲循環、鍵盤監聽與碰撞判定。

## 🚀 如何在本地運行

1.  複製此專案到您的本機目錄。
2.  在專案根目錄下啟動一個簡易的 HTTP 伺服器：
    ```bash
    # 使用 Python 啟動
    python -m http.server 8085
    
    # 或使用 Node.js (http-server) 啟動
    npx http-server -p 8085
    ```
3.  在瀏覽器中開啟 `http://localhost:8085` 即可開始遊玩！
